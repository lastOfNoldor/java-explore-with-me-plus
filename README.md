# Модерация событий администратором
_В данной фиче реализованы 2 дополнительных контроллера разделенных по уровню доступа, а также Сервис и Репозиторий для работы и хранения сущностей представляющих комментарии модерации_ 

* Получение событий для модерации происходит по эндпоинту: GET `{baseURL}/admin/events/moderation`
* Модерация события с возможностью оставления комментария происходит эндпоинту: PATCH `{baseURL}/admin/events/{eventId}/moderate` 
* Получение Пользователем комментариев модерации для события происходит по эндпоинту: GET `{baseURDL}/users/{userId}/events/{eventId}/moderation-comments`
